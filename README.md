# Projeto de Exemplos - Lógica de Programação

Este repositório contém códigos de exemplo utilizados no curso de Lógica de Programação.

## Objetivo

Os arquivos presentes neste projeto foram criados para apoiar o aprendizado de conceitos básicos de programação, como:

- Variáveis
- Entrada e saída de dados
- Operações matemáticas
- Estruturas básicas de raciocínio lógico

## Arquivos do projeto

- `helloworld.py`
- `valorhora.py`
- `variaveis.py`

---

Material voltado para estudo e prática durante o curso.
