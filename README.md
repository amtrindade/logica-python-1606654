# Projeto de Exemplos - Lógica de Programação

Este repositório reúne exemplos simples em Python usados para praticar fundamentos de lógica de programação.

## Objetivo

Os arquivos deste projeto foram organizados para apoiar o aprendizado de conceitos básicos, como:

- entrada e saída de dados
- variáveis e tipos primitivos
- operadores de comparação e operadores lógicos
- estruturas condicionais
- laços de repetição
- funções
- listas e operações com coleções

## Para quem é este projeto

Este material é indicado para quem está começando em programação e quer aprender, na prática, como os comandos básicos do Python funcionam.

Cada arquivo foi separado por assunto para facilitar o estudo passo a passo.

## Estrutura do projeto

Os exemplos estão no diretório src:

- src/helloworld.py: leitura de um nome com input e exibição de mensagem com f-string
- src/valorhora.py: entrada de salário e horas trabalhadas para calcular o valor da hora
- src/variaveis.py: exemplos com string, int, float e bool, além de verificação de tipo com type
- src/operadores-logicos.py: uso de if, elif e else com operadores de comparação e operadores lógicos
- src/maior-valor.py: comparação entre dois números para descobrir qual é o maior
- src/velocidademaxima.py: validação de velocidade com base no limite permitido
- src/chute-valores.py: lógica de tentativa com comparações para acertar um valor
- src/loop-for.py: repetição com for e range
- src/loop-while.py: repetição com while baseada em condição
- src/loop-lista.py: percurso de listas, soma de valores e contagem de itens
- src/soma-multiplica-lista.py: operações de soma e multiplicação com elementos de lista
- src/funcoes.py: criação de funções para soma, média entre dois valores e média de uma lista
- src/fatorial.py: cálculo de fatorial com abordagem iterativa e recursiva

## O que observar em cada exemplo

- Nos arquivos com input, observe que os valores digitados pelo usuário chegam como texto e muitas vezes precisam ser convertidos com int ou float.
- Nos exemplos com if, observe que o Python avalia condições como verdadeiras ou falsas para decidir qual bloco executar.
- Nos exemplos de laço, observe como uma variável muda de valor a cada repetição.
- Nos exemplos com lista, observe que o programa percorre cada item da coleção em sequência.
- Nos exemplos com funções, observe como uma tarefa pode ser definida uma vez e reutilizada depois.

## Como executar

Execute qualquer exemplo com Python 3 a partir da raiz do projeto:

```bash
python3 src/helloworld.py
```

Troque o nome do arquivo conforme o exemplo que deseja testar.

Alguns arquivos pedem valores digitados no terminal. Nesses casos, basta informar o valor solicitado e pressionar Enter.

## Sugestão de prática

Uma forma boa de estudar este projeto é executar um arquivo, tentar prever o resultado e depois alterar pequenos trechos para observar o comportamento do programa.

Exemplos de prática:

- trocar valores numéricos e verificar como o resultado muda
- alterar condições de if e elif
- mudar os itens de uma lista
- criar novas funções a partir dos exemplos existentes
